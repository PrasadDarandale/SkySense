# SkySense
SkySense: Wheather Dashboard
SkySense is a smart, interactive dashboard that delivers real-time weather and air quality insights through clear visuals, helping users quickly understand atmospheric conditions and trends.

This project is an interactive weather and air quality dashboard designed to provide real-time environmental insights in a clean, modern, and user-friendly interface. The dashboard presents current weather conditions, short-term forecasts, and detailed air quality metrics for better situational awareness and decision-making.
